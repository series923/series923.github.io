How Dark Mode Works: A Guide to HTML, CSS, and JavaScript

August 20, 2025

This website has a dark mode toggle button in the top right corner. This feature is enabled through the cooperation of HTML, CSS, and JavaScript.

We use **CSS variables** (`--...`) to define colors and then override them with a `.dark-mode` class. Then, we use JavaScript's `classList.toggle()` to add or remove this class from the `` tag when the button is clicked.

By using this simple mechanism, you can easily create various theme toggles. If you're interested, give it a try on your own website!
